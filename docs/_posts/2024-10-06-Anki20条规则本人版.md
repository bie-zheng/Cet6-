# Anki20条规则本人版

# 不理解的内容别加入

学数学的时候，算出来一特好的结果，开始复习，复习着复习着感觉不对，这玩意怎么看都有别的问题，最后搁置了，因为确实没搞懂，而且似乎也不重要。

专门建立一个文档收集这种东西可能更好一点

加入到记忆里面的东西要谨慎一点，也防止做了太多垃圾卡导致复习不过来。

# 制卡不要太快

在下本科专业是学物理的，吸取的一个教训是，你学的越近的知识点，你就越感觉它很重要，所以经常痛苦面具于这啥几把卡，这么rz的卡我是怎么搞出来的。

# 少图省事用Cloze模板（挖空模板）

多用Basic，简简单单问答模板就行，加深对知识的逻辑关系的认识。Cloze模板用多了就是屎。

# 不要用多个牌组复习

用一个统一的大牌组来复习，别的牌组可以是临时做的卡，或者等待修改的卡，复习只复习一个牌组。

多个牌组复习没有时间

# 渐进改卡

关于改卡，怎么说呢，实际上没用的卡有很多，如果你坚持把它们复习完，那就相当于吃了一大坨，然后发现复习时间到了，还有一大坨。

所以改卡很重要，注意是直接在ANKI的卡组浏览界面改卡不是在复习的时候，按照复习改卡太慢了。微信读书导出的笔记是不能直接用的，必须改，每天改十张卡就行了，很多观点都是重复的内容你会反复摘录，最后发现他们的意思都差不多。

# 标签

来源使用标签标注，标签是支持多级标签的，而且每次添加完新卡之后标签也是不会消失的，所以标签就是整来源的，而且也方便管理。

# 不要想着和上课兼容

Anki这种free study类型的软件就是得投入很多精力，因为我体会到的大学教育完全是忙的一批，没那个时间。现在我开始有自己的时间了，才开始复习各个科目制卡，读自己喜欢的教材。

有老师上课，听他们讲课就是折磨了，不是说讲的不好，是系统性原因，不会一点就都不会了，听懂又感觉讲的太慢感到厌烦，这种debuff的环境下真的很难有剩余的精力（对我来说），哪里有心情再回来搞ANKI。

任何课堂教育都不是为了某个人服务的。

在课堂教育上获取高分最好的方法就是，以老师为中心，让老师的意志和思想覆盖你的意志，听老师讲的内容，学习老师安排的知识点，尤其是考试重点，这种东西不听课不行的。自学的没什么重点，没什么计划，没有啥学啥不学的安排，和考试对不齐，一般下场不好；再说有的人真不是自学，是搁那玩呢。最后考前突击一下，把课后题再做一做，有social比较好的，找学长学姐把去年的考试题拿来，如果老师不透题，那么去年考试题很可能就是没怎么大改，然后基本上有一个不错的成绩。

虽然整个过程是痛苦的，就像被强奸一样，但是最终满脸不明液体的我看着保研的资格泪欣慰的流了出来，阿里嘎多学校桑，给了我一张人生优惠券（限定特款）。

# 开心就好，不必每天都复习

如题，给太大压力容易让自己感到郁郁，烦躁，然后不玩了。

没那么大压力反而会好一些。
